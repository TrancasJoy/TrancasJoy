# TrancasJoy

Site estático simples para divulgar o trabalho de Trancas Joy.

Como usar

1. Personalize `index.html` com seu texto, telefone e e-mail.
2. Substitua as imagens em `assets/images/` (ou altere os links na `index.html`).
3. Faça commit e push para a branch `main` — o workflow em `.github/workflows/deploy.yml` publicará automaticamente no GitHub Pages.

Observações

- Configure o GitHub Pages nas configurações do repositório para usar a branch `gh-pages` gerada pelo workflow, ou verifique se o Pages está ativado para a pasta `/ (root)`.
- Se quiser, eu posso: adicionar formulário de contato, integração com WhatsApp, ou transformar isso em um site responsivo mais completo.
